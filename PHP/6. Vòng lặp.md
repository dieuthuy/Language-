- **foreach**: cho phép duyệt qua tất cả các giá trị của mảng mà không cần kiểm tra điều kiện.

==> **Cú pháp: `foreach(mảng as biến){}`**

*EX*: 

```

<?php
  $cauchao = array("hello", "xinchao","hi");
  foreach ($cauchao as $giatri)
  {
    echo $giatri;
  }
?>
```
